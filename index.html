À propos de cet outil

🎓
Audience

Étudiants en sciences islamiques (ṭullāb al-ʿilm) et chercheurs. L'outil suppose un intérêt sérieux pour les sources chiites primaires.

🔑
Clé API Anthropic

Cet outil nécessite une clé API personnelle obtenue sur console.anthropic.com. Elle est stockée uniquement dans votre navigateur (localStorage) et part directement vers l'API Anthropic — elle n'est jamais transmise à aucun serveur tiers.

⚖️
Double-vérification

L'IA applique un protocole de double-vérification et marque ⚠️ tout élément incertain. Les numéros de page et de volume doivent être confirmés dans les sources primaires avant tout usage académique sérieux.

🌐
Sources de référence

thaqalayn.net · wasail-al-shia.net · al-islam.org · lib.eshia.ir · sistani.org · wikishia.net

🚫
Ce que cet outil n'est pas

Ni autorité religieuse ni source de fatwā. Pour les questions de fiqh pratique, consultez votre marjaʿ qualifié (Sistānī, Khāmeneʾī, etc.).
